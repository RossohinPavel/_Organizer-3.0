###################################### Добавление текстового шаблона ######################################

    Для добавления текстового шаблона в программу создайте в текущем каталоге (/data/mail_samples) 
текстовый документ (в формате txt). В файле нужно явно использовать кодировку UTF-8 для правильного 
отображения кириллических символов. Для простоты, можно скопировать текущий файл (__readme.txt), 
переименовать его и отредактировать текст соответствующим образом.
    Можно дать любое поддерживаемое операционной системой название для текстового файла. Если в начале 
текста шаблона не будет указан атрибут #header, то имя файла будет использовано как название шаблона.
Атрибут #tag определяет таг, по которому шаблоны будут сортироваться в программе.

############################################## Использование ##############################################

    Выбираем шаблон в окне и жмем кнопку - Использовать. Текст из шаблона будет помещен в буфер обмена, 
откуда его можно будет вставить в письмо, мессенджер и т.д.

########################################### Переменные элементы ###########################################

    Текстовые шаблоны поддерживают переменные элементы. Вместо них программа даст подставить какую-нибудь
информацию. В тексте шаблона эти элементы выделены литералом ?% (знак вопроса и процента в начале и в 
конце слова). При инициализации шаблона программа даст подсказку, что подставить вместо этого элемента. 
Если значение не было внесено в соответствующее поле, то будет подставлена пустая строка, а шаблон будет 
удален. Для правильности работы шаблона использовать такую комбинацию символов в других целях нельзя. 
Для добавления переменного элемента выделите слово соответствующим литералом по образцу, указанному в примере.

################################################## Пример #################################################
#header=Приветствие
#tag=Общие
?%Имя?%, здравствуйте.
Вас беспокоит ФотокнигиОптом, Киров.
По поводу заказа ?%заказ?%.
